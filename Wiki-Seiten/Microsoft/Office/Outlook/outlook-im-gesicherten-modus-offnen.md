# Outlook im gesicherten Modus öffnen

Wenn getestet werden will ob Outlook ohne Addons funktioniert, dann folgendes in der Commandokonsolen eingeben:

Abgesicherter Modus

```
outlook.exe /safe
```

Outlook ohne AddOn´s starten

```
outlook.exe /safe:3
```
