# SCOM: Server 2008 100% CPU Auslastung

Auf Windows Vista, XP SP3 oder Server 2008 kann es in abständen vorgekommen das die CPU Auslastung der **MonitoringHost.exe** auf 100% steigt.

  
Lösung:  
Installieren des KB968967
