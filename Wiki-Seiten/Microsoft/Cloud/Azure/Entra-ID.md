# Sicherheit
## Identität
### Abschalten das Benutzer für alle Applikationen Unternehmensdaten freigeben dürfen
Als erstes muss im [Entra Portal](https://entra.microsoft.com) unter den Unternehmensanwendungen, die **"Einwilligung und Berechtigungen"** abgeändert werden.
<img src="/media/entra-id-einwilligung-001.png" width="80%">

Unter diesem Punkt stellt man es mindestens auf die empfohlene Einstellung
<img src="/media/entra-id-einwilligung-002.png" width="80%">

Im Unterpunkt für die Berechtigungsklassifizierung kann man niedrig eingestufte Berechtigungen für den Benutzer freigeben. Beim ersten mal sollten Vorschläge dafür vorhanden sein. Ansonsten hier eine Übersicht.
<img src="/media/entra-id-einwilligung-003.png" width="80%">
